Les principaux raccourcis clavier pour Linux Ubuntu

  • Ctrl+Alt+T : Ouvrir un Terminal
  • Ctrl + Alt + L : Verrouiller la session
  • Ctrl + Alt + Flèche : Changer de bureau
  • Ctrl + F : Rechercher depuis la fenêtre
  • Alt + Tab : Changer de fenêtre
  • Ctrl + W : Fermer onglet actuel
  • Ctrl + S : Sauvegarder
  • Ctrl + N : Ouvrir nouvel onglet (Atom + Gedit) / Nouvelle fenêtre (Mozilla)
  • Ctrl + T : Nouvel onglet
  • Ctrl + c : copie la selection
  • Ctrl +x : coupe la selection
  • Ctrl + v : colle la selection
  • Alt + f4 : ferme la fenetre du logiciel
